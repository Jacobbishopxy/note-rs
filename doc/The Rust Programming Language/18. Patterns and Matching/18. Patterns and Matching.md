# 模式和匹配

模式是一种特殊的 Rust 语法用于匹配类型中的结构，无论类型是简单还是复杂。结合使用模式和`match`表达式以及其它结构可以提供更多对程序控制流的支配权。模式由以下组合构成：

- 字面值
- 结构的数组、枚举、结构体或元组
- 变量
- 通配符
- 占位符

这些组件描述了我们需要处理的数据的形状，接着我们可以匹配其值来决定程序是否拥有正确的数据来继续运行特定部分的代码。
